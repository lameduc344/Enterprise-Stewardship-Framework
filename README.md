# Enterprise Stewardship Framework (ESF)

**Version:** 0.1.0  
**Established:** June 19, 2026  
**Steward:** Nun Nagi El / Mark & Lee Consulting

The Enterprise Stewardship Framework is a practical system for governing an intellectual and economic estate through enterprise health, portfolio leverage, risk awareness, capital allocation, operational controls, and adaptive execution.

## Core Purpose

To create, protect, grow, and transfer intellectual, financial, relational, organizational, and cultural capital across time.

## Working Thesis

A healthy enterprise consistently advances its purpose, sustains itself through changing conditions, and adapts without losing alignment, accountability, resilience, or relevance.

## Minimum Viable Framework

1. Stewardship Principles
2. Enterprise Classification
3. Portfolio Leverage
4. Enterprise Health
5. Risk and Control Registers
6. Stewardship Review
7. AI Officer System

## Portfolio First

ESF treats ventures as parts of an estate rather than isolated projects.

The governing question is not:

> Which project do I love most?

It is:

> Which enterprise creates the greatest value for the entire portfolio within the present stewardship horizon?
